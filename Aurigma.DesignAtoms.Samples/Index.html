Design Atoms Framework

Quick Start

This is a quick start app for Design Atoms Framework

Basic Back-End Interaction Sample

Demonstrates how to request a product from server and render the result.

Vector mask Sample

Demonstrates how you can apply vector mask to design elements.

Curved Text Sample

Demonstrates how to create, load from back-end, and render curved text.

OpenType Features Sample

Demonstrates how you can work with OpenType features.

Viewer Events Sample

Demonstrates how you can handle events from the canvas and design elements.

Pdf Box Back-End

Demonstrates how you can specify safety lines at server side.
